The Problem with Regular Windows

The roof-wall boundary is SPLIT between windows. Pixels at row 3 (bottom of Window 0) cannot see row 4 (top of Window 2).

The chimney top and the wall below it are in different windows -- the model sees them as unrelated!

The Fix: Shifted Windows

In the next layer, windows shift by M/2 pixels. Now boundary pixels CAN see across.

The roof connects to the wall. The chimney connects to the roofline. New connections (shown in gold) bridge what was previously split.
M=4: Each pixel attends to 16 neighbors in its window. Global attention: each pixel would see ALL 64 pixels but costs 64x more compute. Swin gets global coverage in 2 layers (regular + shifted) at a fraction of the cost.
