Why does this help? Without BatchNorm, the network sees wildly different brightness distributions each batch — one batch might be all dark photos, the next all bright. The network wastes effort adapting to these shifts instead of learning useful features. With BatchNorm, the distribution is always stable and centered, so the network can focus on what actually matters in the images.
